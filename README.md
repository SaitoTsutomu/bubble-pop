# Bubble Pop

## 概要

制限時間内に泡をクリックして割り、スコアを稼ぐ1ページゲームです。

タイトル画面で写真をアップロードして解析すると、写真の特徴から元素が推定されます。
推定された元素の原子番号が、ゲーム中の得点とバブル表記に反映されます。

AI実況をONにすると、ゲーム中に一言アドバイス実況が表示されます。
対応環境ではブラウザ内ローカルLLM（Transformers.js）で生成し、未対応環境や失敗時はフォールバック実況に自動切り替えされます。

## 実行方法

`index.html` をブラウザで開くだけで動作します。

ただし AI実況のローカルLLM機能を使う場合は `file://` ではなく `http://localhost:8555` 経由で開いてください。

例（このフォルダで実行）:

```bash
python3 -m http.server 8555
```

その後、`http://localhost:8555` を開きます。

## 操作

- マウスまたはタップで泡を割る
- タイトル画面で写真を選択し「写真を解析する」を押すと元素セットを変更できる
- 通常バブルには `元素記号 + 原子番号` が表示され、割るとその原子番号が加点される
- 同じ元素を連続で割ると `CHAIN` 演出が発生する
- 赤い危険バブルを押すと無得点で画面中の泡が全消去される
- タイトル画面の `AI実況` をONにすると、プレイ中に状況に応じた短文実況が表示される
- 30秒経過で終了

## ファイル構成

- `index.html`: UIとレイアウト
- `styles.css`: 見た目とアニメーション
- `script.js`: ゲームロジック
